 🚀 Placement Readiness Analyzer

An AI-based resume and job description analysis tool built using Streamlit and Google Gemini AI.


📌 Features

- 📄 Resume Parsing (PDF + OCR)
- 🎯 Skill Matching with Job Description
- 📊 ATS Score Calculation
- ❌ Skill Gap Detection
- 🗓️ Personalized Learning Timeline
- 🤖 AI Feedback using Gemini
- 📥 Downloadable Reports (CSV & PDF)
- 📊 Skill Radar Visualization


 🛠️ Technologies Used

- Python
- Streamlit
- Plotly
- Google Gemini AI
- PDF Processing Libraries


 ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
